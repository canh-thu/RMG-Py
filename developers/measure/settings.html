11. rmgpy.measure.settings - Execution settings¶

Contains a number of settings to be used throughout MEASURE. These can be modified at runtime.

rmgpy.measure.settings.minimizeDensityOfStatesCalculations = False¶

Whether or not the number of density of states calculations should be minimized
