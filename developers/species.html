rmgpy.species — Chemical Species¶

This module contains classes and functions for working with chemical species.

From the IUPAC Compendium of Chemical Terminology, a chemical species is “an ensemble of chemically identical molecular entities that can explore the same set of molecular energy levels on the time scale of the experiment”. This definition is purposefully vague to allow the user flexibility in application.

In RMG Py, a chemical species – a local minimum on a potential energy surface – is represented in memory as a Species object. This module also contains the TransitionState class for representing chemical reaction transition states (first-order saddle points on a potential energy surface).

Species¶

class rmgpy.species.SpeciesError¶

An exception class for exceptional behavior that occurs while working with chemical species. Pass a string describing the circumstances that caused the exceptional behavior.

class rmgpy.species.Species¶

A chemical species, representing a local minimum on a potential energy surface. The attributes are:

Attribute Type Description
index int A unique nonnegative integer index
label str A descriptive string label
thermo ThermoModel The thermodynamics model for the species
states StatesModel The molecular degrees of freedom model for the species
molecule list The Molecule objects describing the molecular structure
E0 Quantity The ground-state energy
lennardJones LennardJones A set of Lennard-Jones collision parameters
molecularWeight Quantity The molecular weight of the species
collisionModel object The collisional energy transfer model to use
reactive bool True if the species participates in reactions, False if not
fromAdjacencyList()¶

Load the structure of a species as a Molecule object from the given adjacency list adjlist and store it as the first entry of a list in the molecule attribute. Does not generate resonance isomers of the loaded molecule.

generateResonanceIsomers()¶

Generate all of the resonance isomers of this species. The isomers are stored as a list in the molecule attribute. If the length of molecule is already greater than one, it is assumed that all of the resonance isomers have already been generated.

isIsomorphic()¶

Return True if the species is isomorphic to other, which can be either a Molecule object or a Species object.

toAdjacencyList()¶

Return a string containing each of the molecules’ adjacency lists.

Transition States¶

class rmgpy.species.TransitionState¶

A chemical transition state, representing a first-order saddle point on a potential energy surface. The attributes are:

Attribute Type Description
label str A descriptive string label
states StatesModel The molecular degrees of freedom model for the species
E0 Quantity The ground-state energy in J/mol
frequency Quantity The negative frequency of the first-order saddle point in cm^-1
degeneracy int The reaction path degeneracy
