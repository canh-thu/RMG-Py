5. The rmg.io Module¶

5.1. Classes in rmg.io¶

5.1.1. InvalidInputFileException¶

class rmg.io.InvalidInputFileException(msg)¶
An exception used when parsing an RMG input file to indicate that the input file is invalid. The msg parameter is used to specify what about the file caused the exception to be raised.

5.2. Methods in rmg.io¶

rmg.io.getElements(parent, name)¶
Returns a list of all child elements of the XML parent element with a matching name. The parent parameter comes from a node of a DOM tree as produced via the xml.dom.minidom module.
rmg.io.getElementText(element)¶
Returns the string text that is found between the open and close tags of element, or an empty string if no text is found. The element parameter comes from a node of a DOM tree as produced via the xml.dom.minidom module.
rmg.io.getFirstChildElement(parent, name)¶
Returns the first child element of the XML parent element with a matching name. The parent parameter comes from a node of a DOM tree as produced via the xml.dom.minidom module.
rmg.io.readInputFile(fstr)¶
Parse an RMG input file at the location fstr. If successful, this function returns a rmg.model.CoreEdgeReactionModel object and a list of one or more rmg.model.ReactionSystem objects.
rmg.io.writeOutputFile(fstr, reactionModel, reactionSystems)¶
Write an RMG output file at the location fstr based on the provided reactionModel and reactionSystems.
