4.1. The rmg.unirxn.io Module¶

This module handles the reading and writing of input and output files involving standalone calls to the rmg.unirxn module. Functions in this module utilize the rmg.io.XML class to work with the input and output XML data.

4.1.1. Methods in rmg.unirxn.io¶

rmg.unirxn.io.readInputFile(fstr)¶
Parse an input file found at fstr. If successful, this function returns a Network object containing the unimolecular reaction network, lists of temperatures in K and pressures in Pa, the maximum grain size in J/mol, the minimum number of grains, the approximate method to use, and the interpolation model to fit.
rmg.unirxn.io.writeInputFile(fstr, network, Tlist, Plist, Elist, method, model)¶
Write an input file to fstr. Parameters are: network, a Network object containing the unimolecular reaction network; Tlist, a list of temperatures in K; Plist, a list of pressures in Pa; Elist, a list of energy grains in J/mol; method, the approximate method to use; and model, the interpolation model to fit.
