4.4. The rmg.unirxn.network Module¶

Contains classes that define an internal representation of a unimolecular reaction network.

4.4.1. Classes in rmg.unirxn.network¶

4.4.1.1. Isomer¶

class rmg.unirxn.network.Isomer(species=None)¶

An isomer well or product channel of a unimolecular reaction network. This represents a single local minimum on the corresponding potential energy surface, and is composed of one or two chemical species. The attributes are:

Attribute Description
species A list of the species that make up the isomer; items in the list are Species objects
E0 The ground-state energy of the isomer
collFreq For unimolecular isomers, the collision frequency in s^-1
densStates The density of states for the isomer in mol/J
eqDist The equilibrium distribution for the isomer
Q The partition function for the isomer

The entries in the lower half of the above table are used for temporary storage while conducting a pressure-dependent rate coefficient calculation.

calculateCollisionFrequency(T, P, bathGas)¶
Calculate the collision frequency for the isomer (unimolecular only) at the given temperature T in K, pressure P in Pa, and using species bathGas as the bath gas.
calculateDensityOfStates(Elist)¶
Calculate the density of states in mol/J of the isomer at the specified list of energies Elist in J/mol.
calculateEqDist(Elist, T)¶
Calculate the equilibrium (Boltzmann) distribution of the isomer using the stored density of states which corresponds to the list of energies Elist in J/mol and at the temperature T in K. The partition function at that temperature is also determined. The equilibrium distribution is not returned, but is instead stored in the eqDist attribute.
clean()¶
Free some memory by deleting temporary variables used only during a single phenomenological rate coefficient estimation. This should be called at the end of that calculation.
isMultimolecular()¶
Return True if the isomer represents a multimolecular reactant or product channel or False otherwise.
isUnimolecular()¶
Return True if the isomer represents a unimolecular isomer well or False otherwise.

4.4.1.2. Network¶

class rmg.unirxn.network.Network(id=0)¶

A representation of a unimolecular reaction network. The attributes are:

Attribute Description
id A unique identifier for the network
isomers A list of Isomer objects that make up the network
pathReactions A list of Reaction objects that connect adjacent isomers (the “path” reactions)
netReactions A list of Reaction objects that connect any pair of isomers (the “net” reactions)
valid True if the net reaction kinetics are up to date; False if the kinetics are out of date and need to be recomputed
explored A list of all of the fully-explored unimolecular isomers
addPathReaction(rxn)¶
Add the Reaction object rxn to the network as a path reaction.
applyApproximateMethod(T, P, Elist, method, errorCheck=True)¶
Apply the approximate method specified in method to estimate the phenomenological rate coefficients for the network. This function expects that all preparations have already been made, as in the calculateRateCoefficients() method.
calculateDensitiesOfStates(Elist)¶
Calculate the density of states for all isomers in the network whose species have spectral data. This is required for all unimolecular isomers and for all multimolecular isomers that are reactants of a path reaction with the forward reaction defined as the association.
calculateRateCoefficients(Tlist, Plist, Elist, method, errorCheck=True)¶
Calculate the phenomenological rate coefficients for the network.
containsSpecies(species)¶
Return True if the species is a unimolecular isomer in the network, and False if not.
determineEnergyGrains(grainSize, numGrains, Tmax)¶
Select a suitable list of energies to use for subsequent calculations. The procedure is to (1) calculate the equilibrium distribution of the highest-energy isomer at the largest temperature of interest (to get the broadest distribution), (2) calculate the energy at which the tail of the distribution is some fraction of the maximum, and (3) add the difference between the ground-state energy of the isomer and the highest ground-state energy in the system (either isomer or transition state). Parameters are the desired grain size grainSize and number of grains numGrains and the temperature to use for the equilibrium calculation Tmax, which should be the highest temperature of interest.
getEnergyGrains(Emin, Emax, dE0, nGrains0)¶
Return an array of energy grains that have a minimum of Emin, a maximum of Emax, and either a spacing of dE0 or have number of grains nGrains0. The first three parameters are in J/mol.
getLeakFlux(T, P, conc, totalConc=None)¶
Return the leak flux of the network: the forward flux to all unexplored unimolecular isomers in the network.
getMaximumLeakSpecies()¶
Get the unexplored (unimolecular) isomer with the maximum leak flux.
getSpeciesList()¶
Return a list of all of the species in the network (excluding the bath gas), in no particular order.
indexOf(object)¶
Return the integer index associated with a given object, which can be either an instance of Isomer or Reaction. Raises a KeyError if the object is not found in the network; thus you can assume that the returned index is valid.
invalidate()¶
Mark a network as in need of a new pressure-dependence calculation.
merge(other)¶
Merge another Network object other into this object.
numMultiIsomers()¶
Return the number of multimolecular isomers in the network.
numUniIsomers()¶
Return the number of unimolecular isomers in the network.
shiftToZeroEnergy()¶
Adjust the energies of the system (i.e. the E0 attribute of isomers) such that the lowest energy in the system is set to zero.

4.4.2. Exceptions in rmg.unirxn.network¶

4.4.2.1. UnirxnNetworkException¶

exception rmg.unirxn.network.UnirxnNetworkException(msg)¶
An general exception used when an error is encountered while dealing with a unimolecular reaction network. The msg attribute can be used to store a string description of the cause of the exception.
