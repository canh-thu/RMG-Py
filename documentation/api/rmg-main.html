6. The rmg.main Module¶

Contains the main function for RMG execution and several helper functions that support it.

6.1. Methods in rmg.main¶

rmg.main.execute(inputFile, outputDir, scratchDir, libraryDir, verbose)¶
Generate a reaction model for the set of reaction systems specified in an input file at location inputFile. Output and temporary files will be placed in the directories outputDir and scratchDir, respectively; if either of these are empty strings, the corresponding files will be placed in the same directory as the input file. The libraryDir parameter can be used to specify additional directories to search for RMG databases. The verbose parameter is an integer specifying the amount of log text seen at the console; the levels correspond to those of the logging module.
rmg.main.initializeLog(verbose)¶
Set up a logger for RMG to use to print output to stdout. The verbose parameter is an integer specifying the amount of log text seen at the console; the levels correspond to those of the logging module.
rmg.main.printRMGHeader()¶
Output a header containing identifying information about RMG to the log.
