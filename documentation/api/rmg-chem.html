1. The rmg.chem Module¶

Contains classes describing simple chemical entities: elements, atoms, bonds, etc.

1.1. Classes in rmg.chem¶

1.1.1. Atom¶

class rmg.chem.Atom(atomType=None, electronState=None, charge=0, label='')¶

Represent an atom in a chemical species or functional group. The atomType and electronState attributes contain lists of the allowed atom types and electron states, respectively, for this atom. The charge attribute stores the resulting formal charge. The label attribute can be used to tag individual atoms, e.g. center atoms or reactive sites in functional groups.

atomType¶
Get the list of allowed atom types. If the list is of length 1, the lone item in the list is returned instead.
canDecreaseFreeElectron()¶
Return True if the number of unpaired electrons on this atom can be decreased by one. This method requires that there be at least one free electron on this atom.
canIncreaseFreeElectron()¶
Return True if the number of unpaired electrons on this atom can be increased by one. This method restricts the radical order to three or fewer.
copy()¶
Generate a copy of the current atom.
decreaseFreeElectron()¶
Decrease the number of unpaird electrons on this atom by one.
electronState¶
Get the list of allowed electronic states. If the list is of length 1, the lone item in the list is returned instead.
equivalent(other)¶
Return True if self and other are equivalent atoms, i.e. they have the same element and electronic state.
getAtomType()¶
Get the list of allowed atom types. If the list is of length 1, the lone item in the list is returned instead.
getElectronState()¶
Get the list of allowed electronic states. If the list is of length 1, the lone item in the list is returned instead.
getElement()¶
Return the element that this atom represents. If there are multiple atom types, then this method returns None.
getFreeElectronCount()¶
Return the number of unpaired electrons.
hasFreeElectron()¶
Return True if the atom has one or more unpaired electrons and False otherwise.
increaseFreeElectron()¶
Increase the number of unpaired electrons on this atom by one.
isCarbon()¶
Return True if the atom is a carbon atom and False otherwise.
isCenter()¶
Return True if the atom is a center atom and False otherwise.
isElement(symbol)¶
Return True if the atom has the element with the symbol symbol and False otherwise.
isHydrogen()¶
Return True if the atom is a hydrogen atom and False otherwise.
isNonHydrogen()¶
Return True if the atom is not a hydrogen atom and False otherwise.
isOxygen()¶
Return True if the atom is an oxygen atom and False otherwise.
setAtomType(atomType)¶

Set the atom type that this atom represents. The atomType parameter is any of:

  • A string containing the label of a single atom type
  • An AtomType object representing the atom type
  • A list containing one or more of each of the above

In all cases, the data will be stored internally as a list of AtomType objects.

setElectronState(electronState)¶

Set the electron state that this atom represents. The electronState parameter is any of:

  • A string containing the label of a single electron state
  • An ElectronState object representing the electron state
  • A list containing one or more of each of the above

In all cases, the data will be stored internally as a list of ElectronState objects.

1.1.2. AtomType¶

class rmg.chem.AtomType(label, element, description)¶

A type of atom such as Cd, a carbon atom with all single bonds.

Represent a single atom type by its chemical element and, optionally, some information about the local bond structure around that element. Each element has a unique string label, the underlying chemical element, and a string description of the element.

This class is specifically for properties that all atoms of the same element share. Ideally there is only one instance of this class for each element.

equivalent(other)¶
Returns True if two atom types are equivalent or False otherwise.

1.1.3. Bond¶

class rmg.chem.Bond(atoms, bondType='')¶

Represent a bond in a chemical species. Each bond has a list atoms of length two containing the two atoms in the bond and a bondType object, stored internally as a BondType object.

canDecreaseOrder()¶
Return True if the bond order can be decreased by one without breaking.
canIncreaseOrder()¶
Return True if the bond order can be increased by one.
copy()¶
Generate a copy of the current bond.
decreaseOrder()¶
Decrease the bond order by one.
equivalent(other)¶
Return True if self and other are equivalent bonds, i.e. they have the same bond type.
increaseOrder()¶
Increase the bond order by one.
isBenzene()¶
Return True if the bond represents a benzene (aromatic) bond and False otherwise.
isDouble()¶
Return True if the bond represents a double bond and False otherwise.
isSingle()¶
Return True if the bond represents a single bond and False otherwise.
isTriple()¶
Return True if the bond represents a triple bond and False otherwise.
setBondType(bondType)¶
Set the bond type that this bond represents. The bondType parameter can be a BondType object, a number representing the bond order, or a string representing the label of the desired bond type. In all cases bondType will be converted to and stored as a BondType object.

1.1.4. BondType¶

class rmg.chem.BondType(label, name, order, piElectrons, location='')¶

Represent a type of chemical bond. Each bond type has a unique string label; a unique string name; a numeric bond order; an integral piElectrons, the number of pi electrons; and a string location with bond geometry information (i.e. ‘cis’ or ‘trans’).

This class is specifically for properties that all bonds of the same type share. Ideally there is only one instance of this class for each bond type.

equivalent(other)¶
Returns True if two bond types are equivalent or False otherwise.

1.1.5. ElectronState¶

class rmg.chem.ElectronState(label, order, spin)¶

Represent a single free electron state (none, radical, etc.) Each state is defined by a unique string label; the order, or number of free electrons; and a spin multiplicity.

This class is specifically for properties that all free electron states share. Ideally there is only one instance of this class for each free electron state.

equivalent(other)¶
Returns True if two electron states are equivalent or False otherwise.

1.1.6. Element¶

class rmg.chem.Element(number, name, symbol, mass, valence)¶

Represent a single chemical element. Each element has an atomic number, a name, a symbol, an atomic mass, and a valence, a list of the possible numbers of bonds allowed.

This class is specifically for properties that all atoms of the same element share. Ideally there is only one instance of this class for each element.

1.2. Methods in rmg.chem¶

rmg.chem.loadAtomTypes()¶
Loads entries into a dictionary of atom types.
rmg.chem.loadBondTypes()¶
Loads entries into a dictionary of bond types. The dictionary created by this function is always available at rmg.chem.bondTypes.
rmg.chem.loadElectronStates()¶
Loads entries into a dictionary of free electron states. The dictionary created by this function is always available at rmg.chem.electronStates.
rmg.chem.loadElements()¶
Loads entries into a dictionary of elements.
