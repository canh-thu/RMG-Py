2.11. The rmg.settings Module¶

Contains a number of settings to be used throughout RMG.

These can be modified at runtime, for example by reading in a settings file. Their values will be stored in this module.

rmg.settings.drawMolecules¶

Whether to draw pictures of the molecules.

rmg.settings.drawMolecules = False

rmg.settings.generatePlots¶

Whether to make plots of simulations.

rmg.settings.generatePlots = False

rmg.settings.initializationTime¶

Time at which the program execution was started (seconds since the epoch)

rmg.settings.initializationTime = 0.0

rmg.settings.libraryDirectory¶

The directory with libraries in?

rmg.settings.libraryDirectory = .

rmg.settings.outputDirectory¶

The directory used for output of results

rmg.settings.outputDirectory = .

rmg.settings.scratchDirectory¶

The directory used for temporary scratch files

rmg.settings.scratchDirectory = .

rmg.settings.spectralDataEstimation¶

Whether to estimate spectral data for species.

rmg.settings.spectralDataEstimation = False

rmg.settings.unimolecularReactionNetworks¶

Whether to process unimolecular (pressure-dependent) reaction networks.

rmg.settings.unimolecularReactionNetworks = False
