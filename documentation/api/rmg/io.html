2.6. The rmg.io Module¶

2.6.1. Classes in rmg.io¶

2.6.1.1. XML¶

class rmg.io.XML(document=None, path='')¶

A class for manipulating an XML DOM tree as created by the xml.dom.minidom package. The DOM is stored in the document attribute.

cleanup()¶
Clean up a DOM tree by unlinking. This should only be done when you are finished working with the tree.
createAttribute(attributeName, parentElement, attributeValue)¶
Create the attribute attributeName as an attribute of parentElement with the value attributeValue in the DOM tree document.
createElement(elementName, parentElement)¶
Create the element elementName as an immediate child of parentElement in the DOM tree document. The created element is returned.
createQuantity(elementName, parentElement, value, units='')¶
Create an element representing a quantity of a certain value (scalar or list) and units (string) with the name elementName as a child of parentElement.
createTextElement(elementName, parentElement, text)¶
Create the element elementName as an immediate child of parentElement in the DOM tree document. Set the text of the created element to text, which is passed through a string conversion function along the way. The created element is returned.
getAttribute(element, name, required=True, default=None)¶
Return the value of the attribute name of element root. If no attribute is found and required is True, an InvalidXMLError is raised. If no attribute is found and required is False, default is returned.
getChildElement(parentElement, name, required=True)¶
Return the child element of root with the name name. If no suitable child element is found, None is returned if required is False or an InvalidXMLError is raised if required is True.
getChildElementText(parentElement, name, required=True, default=None)¶
Return the text of the child element of root with the name name. If multiple child elements are found, no text is found, or no child element is found and required is True, an InvalidXMLError is raised.
getChildElements(parentElement, name, required=True)¶
Return a list of child elements of root with the name name. If no suitable child elements is found, an an InvalidXMLError is raised if required is True.
getChildQuantity(parentElement, name, required=True, default=0.0)¶

Process an element of the form

<elementName units=”units”>value</elementName>

and return the corresponding quantities.Quantity object.

getElementText(element)¶
Return the text of the element element. If no text is found, an InvalidXMLError is raised.
getQuantity(element)¶

Process an element of the form

<elementName units=”units”>value</elementName>

and return the corresponding quantities.Quantity object.

getRootElement()¶
Return the root element of the DOM tree.
initialize(elementName)¶
Initializes a new DOM tree with the root element having the name elementName. Returns the root element.
load(path)¶
Load a DOM from an XML file located at path.
save(path)¶
Save a DOM to an XML file located at path.

2.6.2. Methods in rmg.io¶

rmg.io.loadFrequencyDatabase(dstr)¶
Load the RMG thermo database located at dstr into the global variable rmg.spectral.data.frequencyDatabase.
rmg.io.loadKineticsDatabase(dstr)¶
Load the RMG kinetics database located at dstr into the global variable rmg.reaction.kineticsDatabase.
rmg.io.loadThermoDatabase(dstr)¶
Load the RMG thermo database located at dstr into the global variable rmg.species.thermoDatabase. Also loads the forbidden structures into rmg.thermo.forbiddenStructures.
rmg.io.readDatabaseList(xml0, rootElement)¶
Parse the portion of the RMG input file that tells where the RMG database(s) are that should be used, i.e. the <databaseList> element.
rmg.io.readInputFile(fstr)¶
Parse an RMG input file at the location fstr. If successful, this function returns a rmg.model.CoreEdgeReactionModel object and a list of one or more rmg.model.ReactionSystem objects.
rmg.io.writeOutputFile(fstr, reactionModel, reactionSystems)¶
Write an RMG output file at the location fstr based on the provided reactionModel and reactionSystems.

2.6.3. Exceptions in rmg.io¶

exception rmg.io.InvalidInputFileException(msg)¶
An exception used when parsing an RMG input file to indicate that the input file is invalid. The msg parameter is used to specify what about the file caused the exception to be raised.
exception rmg.io.InvalidXMLError(msg='')¶
An exception to be raised when invalid XML is encountered. The exception should be passed a string msg containing details about the error.
